# Generador de Contraseñas Seguras

## Introducción

El presente proyecto consiste en el desarrollo de una aplicación web denominada **Generador de Contraseñas Seguras**, creada utilizando HTML, CSS y JavaScript. Su propósito es proporcionar una herramienta que permita generar contraseñas aleatorias y seguras, personalizadas según las preferencias del usuario. La aplicación cuenta con una interfaz sencilla e intuitiva que facilita su uso y demuestra la aplicación práctica de tecnologías de desarrollo web.

## Descripción del problema

En la actualidad, muchas personas utilizan contraseñas débiles, repetidas o fáciles de adivinar, lo que incrementa el riesgo de sufrir accesos no autorizados a sus cuentas personales. Además, crear contraseñas seguras de manera manual puede resultar complicado para algunos usuarios. Por esta razón, se desarrolló un sistema que automatiza la generación de contraseñas fuertes, reduciendo la posibilidad de utilizar combinaciones inseguras y promoviendo mejores prácticas de seguridad informática.

## Relación con los contenidos de la asignatura

Se utilizaron **HTML** para la estructura de la página, **CSS** para el diseño y presentación visual, y **JavaScript** para implementar la lógica de programación, la validación de datos y la generación aleatoria de contraseñas. ## Explicación del sistema desarrollado

El sistema permite al usuario generar contraseñas personalizadas mediante una serie de opciones configurables.

Sus principales características son:

* Seleccionar una longitud de entre **4 y 50 caracteres**.
* Incluir letras mayúsculas (A-Z).
* Incluir letras minúsculas (a-z).
* Incluir números (0-9).
* Incluir símbolos especiales (!, @, #, $, %, &, entre otros).

### Funcionamiento del sistema

1. El usuario selecciona la longitud deseada de la contraseña.
2. Marca los tipos de caracteres que desea incluir.
3. El sistema verifica que exista al menos una opción seleccionada.
4. JavaScript genera una combinación aleatoria de caracteres según los criterios establecidos.
5. La contraseña generada se muestra en pantalla.
6. El usuario puede copiar la contraseña para utilizarla en sus diferentes cuentas o servicios.
