Generador de Contraseñas Seguras
Este proyecto consiste en una aplicación web desarrollada con HTML, CSS y JavaScript cuyo objetivo es generar contraseñas seguras y personalizadas según las preferencias del usuario.
La aplicación permite al usuario seleccionar la longitud de la contraseña, pudiendo elegir desde una cantidad mínima de caracteres hasta una cantidad mayor para incrementar la seguridad. Además, ofrece diferentes opciones de personalización mediante casillas de selección.
Entre las opciones disponibles se encuentran:
Incluir letras mayúsculas (A-Z).
Incluir letras minúsculas (a-z).
Incluir números (0-9).
Incluir símbolos especiales (!, @, #, $, %, &, etc.).
Una vez configuradas las opciones deseadas, el usuario puede presionar el botón Generar Contraseña, y el sistema creará automáticamente una contraseña aleatoria basada en los criterios seleccionados.
Funcionamiento del sistema
El usuario ingresa la longitud deseada de la contraseña.
Selecciona los tipos de caracteres que desea incluir.
El programa valida que exista al menos una opción seleccionada.
JavaScript genera una combinación aleatoria de caracteres según los criterios elegidos.
La contraseña se muestra en pantalla.
El usuario puede copiar la contraseña generada para utilizarla en sus cuentas o servicios.
Objetivos del proyecto
Facilitar la creación de contraseñas seguras.
Reducir el uso de contraseñas débiles o fáciles de adivinar.
Mostrar la aplicación práctica de HTML, CSS y JavaScript en el desarrollo web.
Proporcionar una interfaz sencilla e intuitiva para cualquier usuario.
Tecnologías utilizadas
HTML: estructura de la página web.
CSS: diseño visual y estilos de la interfaz.
JavaScript: lógica de generación aleatoria de contraseñas y validación de datos.
